Double : 64비트 부동소수점

 - when

<pre><code>val x = 1

when(x) {
	1 -> println("x==1")											// 값 하나
	2,3 -> println("x==2 or x==3")							// 여러 값은 콤마로

6.for
>자바의 foreach 와 비슷

in 키워드를 사용하여 모든 요소를 num 변수로 가져옴

<pre><code>val numbers = arrayOf(1, 2, 3, 4, 5)

for(num in numbers) {
	print ln(num)		// 1; 2; 3; 4; 5;
}
</pre></code>

증가 범위는 .. 연산자를 사용.
감소 범위는 downTo 키워드를 사용하며 step 키워드로 증감의 간격을 조절

<pre><code>// 1~3까지 출력
for(i in 1..3) {
	println(i)	// 1; 2; 3;
}

// 0~10까지 2씩 증가하며 출력
for(i in 0..10 step 2) {
	println(i)	// 0; 2; 4; 6; 8; 10
}

// 10부터 0까지 2씩 감소하며 출력
for(i in 10 downTo 0 step 2) {
	println(i)	// 10; 8; 6; 4; 2; 0
}
</pre></code>

<pre><code>var x = 10

while(x > 0) {

}

do {
	x--
	println(x)	// 9; ~ 0;

</pre></code>
