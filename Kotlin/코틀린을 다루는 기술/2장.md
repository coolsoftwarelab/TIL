## 2.1 필드와 변수

```
// 불편
val name: String = "Hello"
// 또는 val name = "Hello"

// 가변
var name = "Hello"
```

### 2.1.3 지연 초기화

`val` 지연 초기화  
`by lazy` 를 사용하면 최초 사용시점에 초기화 한다.
```
println("1")
val name: String by lazy { getName() }  // 또는 val name: String by lazy(::getName)
println("2")
println(name)
println(name)
println(name)

fun getName(): String {
    println("getName...")
    return "Hello"
}
```

val 지연 초기화 실행결과
```
1
2
getName...
Hello
Hello
Hello
```

var 지연 초기화
```
lateinit var name: String
...
name = getName()
```
